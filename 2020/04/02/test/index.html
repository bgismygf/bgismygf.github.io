Hello Hexo

發表於 2020-04-02 更新於 2020-04-03 分類於 Hexo

新建立的部落格，記錄一下建立過程！

什麼是 Hexo？

在官方Hexo 文件中提到：

Hexo 是一個快速、簡單且強大的網誌框架。Hexo 使用 Markdown（或其他標記語言）解析您的文章，並在幾秒鐘內，透過漂亮的主題產生靜態檔案。

安裝需求

  • Node.js （建議版本需不低於8.10，建議使用 Node.js 10.0 及以上版本）
  • Git

安裝後，在終端機或 Git Bash 輸入以下即可安裝 Hexo。

建立

Hexo 安裝完成後，執行下列指令，Hexo 會在指定資料夾中建立所有你需要的檔案。

在建立的資料夾裡可以看到以下檔案:

上圖其中的檔案裡，較常使用:

  • scaffolds
  • source
  • themes
  • _config.yml
  • package.json

package.json

安裝的套件或應用程式資訊的記錄，預設安裝的有 EJS, Stylus 和 Markdown renderer，安裝後可刪可留。

其他檔案介紹在後面調整時會講解到。

佈署到 Github 上

題外話，如果想看畫面，可以先用以下指令來啟動 server 來觀看。

啟動後會出現以下，Ctrl + 左點擊即可看到畫面。

畫面陽春些，之後再調整，先 Ctrl + C 退出 server，來接著做佈署。

在 Github 上開新資料庫。

在 Repository name 下方格輸入資料庫名稱。
